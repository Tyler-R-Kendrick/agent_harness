# Manus Gossip

## Positive Signals

- Public discussion recognizes Manus as one of the broadest general-purpose agents, especially because it combines web browsing, files, coding-like work, and asynchronous task completion.
- Browser Operator drew attention because it attacks authenticated browser workflows directly rather than asking users to recreate state in a cloud session.

## Negative Signals

- Reddit users report anxiety about credits being consumed by failed or repeated attempts, especially when tasks require retries or long supervision.
- Some user reports describe Manus claiming success without enough end-to-end verification, getting stuck around downloads or checkpoints, and requiring manual correction.
- Support and pricing complaints are common in public review threads, with users comparing the agent's useful output against the cost of babysitting it.

## Product Risks To Watch

- Browser Operator can improve completion rates while increasing blast radius if approval and audit controls are too coarse.
- Preferred Browser makes recurring automation easier, but it also makes account-state drift and extension-dependent behavior part of the agent runtime.
- If task replay is mostly narrative rather than deterministic evidence, users may still need to manually re-check each important outcome.
