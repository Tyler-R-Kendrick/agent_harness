# Zapier Agents Gossip

## Positive Signals

- Zapier's app ecosystem remains a major trust and convenience advantage.
- Public docs show practical setup flows: instructions, app connections, knowledge sources, Copilot assistance, and usage measurement.
- Users already familiar with Zapier can evaluate agents without adopting a new automation platform.

## Negative Signals

- Reddit discussion around Zapier repeatedly focuses on pricing, task counting, and whether the platform becomes expensive as workflows scale.
- Some users describe agent-style experiments as useful for lightweight tasks but hard to control at larger scale.
- Product documentation warns that Zapier Agents may not currently honor every Enterprise app/action restriction, which is a governance concern.

## Bug And UX Risk Themes

- Activity-based metering can discourage iterative exploration and debugging.
- App-action success does not prove that a browser-facing workflow looked correct to the end user.
- Agent failures can be difficult to distinguish from app permission, knowledge-source, quota, or prompt issues.
