# Automa Gossip

## Positive Signals

- Chrome Web Store shows about 200,000 users and a 4.4 rating across hundreds of ratings at the June 2026 snapshot.
- GitHub scale is strong for the category, with tens of thousands of stars and an active issue/PR queue.
- Third-party review summaries and community references generally praise Automa as reliable, time-saving, free, and flexible for local browser automation.

## Negative Signals

- Open GitHub issues in 2026 include selector failures on ServiceNow pages, iframe click offset problems, loop-data blocks stopping without error logs, URL protocol bugs when variables are used, Firefox sign-in problems, Chrome-version run-button failures, and certificate/login failures.
- Beginner community questions suggest recording and selector concepts are still hard for non-technical users.
- General browser-automation discussions warn that sessions, storage, browser profiles, dynamic pages, and anti-bot behavior make scaling local browser automation messy.

## Bug And UX Risk Themes

- Block-based transparency helps debugging, but it also exposes users to low-level browser failure modes.
- Cross-browser support multiplies compatibility work; Chrome, Firefox, and Edge-like contexts can diverge.
- A free/open-source tool can become a support bottleneck when users expect SaaS-level reliability from community-maintained automation.
