# Microsoft Edge Copilot Mode Gossip

## Positive Signals

- Microsoft has official support guidance for Copilot Actions, including visible tab indicators and advice to avoid sensitive personal or financial tasks.
- Copilot Journeys and history context address a real browsing pain: resuming research without manually reconstructing tab state.
- Edge can ship AI browser features to a large installed base without asking users to migrate.

## Negative Signals

- WindowsCentral reported fresh backlash around Microsoft retiring app-style Sidebar functionality while leaving Copilot untouched, reinforcing user concern that Copilot is crowding out existing Edge features.
- WindowsLatest argued that Edge increasingly feels like Copilot rather than a browser, especially around AI-heavy new tab experiences.
- TechRadar criticized Copilot prompts in Edge as annoying when visiting rival AI products.

## Bug And UX Risk Themes

- "Optional" AI can still feel forced if it occupies the new tab, toolbar, address bar, and sidebar.
- Browser-history AI requires careful opt-in and clear reset controls.
- Copilot Actions need strong guardrails because a wrong click inside a logged-in browser has real-world consequences.
