# Perplexity Comet Gossip

## Positive Signals

- Reddit comments include users calling it a main browser on Windows/Mac, praising research speed and aggregation.
- Users report real value for large repetitive chores such as mailbox cleanup, spreadsheets/forms, and YouTube list cleanup.

## Negative Signals

- Security research from Brave reported indirect prompt-injection issues in Comet and noted incomplete mitigation after retesting.
- User complaints mention random lockups, Android instability, battery drain, missing sync/basic mobile features, slow agent mode, and the agent explaining how to do a task instead of doing it.
- Critics object to the privacy tradeoff of a browser assistant that can observe search, page, and account context.

## Bug And UX Risk Themes

- Reliability varies sharply by workflow complexity.
- Slow autonomous action competes poorly with manual browser work unless the task is long or repetitive.
- Agentic shopping/email flows are compelling demos but dangerous failure surfaces.
