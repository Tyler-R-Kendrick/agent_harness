# Browser Use Cloud Gossip

## Positive Signals

- Public docs emphasize open-source popularity and cloud features such as managed infrastructure, stealth browsers, CAPTCHA solving, residential proxies, and raw CDP.
- The Skills concept is strong because it addresses reliability and latency once a workflow is learned.

## Negative Signals

- GitHub issues show active bugs around tab-list commands, inability to complete tasks with certain GPT versions, Azure OpenAI content filtering on ordinary navigation/login prompts, DOM extraction failures, CDP instability, and browser support requests.
- Pricing has many moving parts, which can create surprise for small teams.

## Bug And UX Risk Themes

- Model/provider variability directly affects browser task reliability.
- DOM extraction and CDP stability are foundational; failures here make agent behavior look flaky even if the prompt is good.
- Governance/audit requests in issues suggest customers need more control before broad production adoption.
