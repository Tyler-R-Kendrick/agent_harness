# BrowserOS Gossip

## Positive Signals

- Public counters emphasize 10k+ GitHub stars, 13 AI providers, 53 browser tools, 40+ MCP integrations, and macOS/Windows/Linux support.
- YC profile reinforces the "privacy-first alternative to Perplexity Comet, Dia" positioning.

## Negative Signals

- GitHub issues show active requests and bugs around scheduled task results, Chromium crashes, MCP tool failures, tab split panes, file/memory attachments, and window/PDF/DOM tool implementation failures.
- Open-source pace is high, but the visible issue list implies a fast-moving product with rough edges.

## Bug And UX Risk Themes

- MCP server stability and tool correctness are critical for developer trust.
- Browser crash reports are especially damaging for a primary-browser replacement.
- The product must make cloud-vs-local model routing unmistakable or privacy messaging can backfire.
