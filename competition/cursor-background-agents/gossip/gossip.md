# Cursor Background Agents Gossip

## Positive Signals

- Reviewers and users often cite Cursor's codebase understanding and multi-file editing as the product's strongest moat.
- Background agents are discussed as a credible answer to long-running coding tasks that should happen away from the foreground editor.
- The IDE handoff is a strength: developers can inspect diffs, continue work, and keep normal Git habits.

## Negative Signals

- Users question whether Bugbot sees enough context and complain about comments that remain outdated instead of resolving cleanly.
- Some users describe model-priced agent usage as harder to budget than a flat subscription.
- Background-agent docs require GitHub app permissions and environment configuration, which can be a barrier for locked-down organizations.

## Bug And Friction Themes

- Stale PR review comments.
- Partial review coverage.
- Usage surprises when high-end models are selected or background tasks are run in parallel.
