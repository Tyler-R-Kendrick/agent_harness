# Apify Gossip

## Positive Signals

- Apify Store claims a large catalog of web scraping and automation tools, with visible usage counts, ratings, and maintained-by labels.
- The AI-agent positioning is credible because Apify already has structured data, APIs, schedules, storage, and marketplace distribution.
- The hosted MCP server gives agents a way to discover and run Actors without each team writing bespoke tool wrappers.

## Negative Signals

- Apify's own AI Web Agent listing says it is experimental and not actively maintained, which weakens its direct browser-agent story even while the broader platform is strong.
- Individual Actor quality varies; public Actor pages expose low ratings, low monthly active users, slow issue response, and open issues for some AI/browser Actors.
- Independent pricing commentary highlights a common buyer confusion: platform plan credits and per-Actor pricing are separate layers.

## Bug And UX Risk Themes

- Long-running AI extraction Actors can be slow or expensive because LLM calls, browser runtime, and platform compute compound.
- Marketplace choice overload can make it hard for an agent or user to choose the right Actor and trust its maintenance state.
- MCP access expands distribution but also makes accidental spend, telemetry expectations, auth, and rate limits part of the user experience.
