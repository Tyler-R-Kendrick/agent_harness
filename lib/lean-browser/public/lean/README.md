# Lean Browser Assets

Host applications should serve Lean 4 browser assets from their own static route, usually `/lean`.

Expected files commonly include:

- `lean_js_wasm.js`
- `lean_js_wasm.wasm`
- `lean_js_js.js`
- `library.zip`
- `library.info.json`
- `library.olean_map.json`

This directory is a placeholder and documentation surface only. Do not commit large generated Lean artifacts here unless the repository policy explicitly allows it.
