# Opera

## Overview
Aria assistant rebuilt as Opera AI in late 2025. Multi-model support with OpenAI and Google endpoints. Serves 100M+ Opera Mini users on mobile and desktop. Not yet fully agentic—does not autonomously act on pages. Cross-platform presence (desktop, gaming, mobile).

## Technical Approach
- **Base**: Chromium
- **AI Integration**: Multi-model assistant (OpenAI, Google) without autonomous page actions
- **Key Differentiator**: Massive mobile user base (100M+ Opera Mini); multi-model flexibility

## Key Learnings
- Mobile-first positioning requires different agentic UX approach than desktop
- Multi-model strategy allows switching and redundancy vs single-vendor lock-in
- 100M+ existing users provide distribution advantage but also legacy constraints
- Non-agentic assistant still valuable but less differentiated

## UX Innovations
- Multi-model assistant selection
- Cross-platform consistency across desktop/mobile
- Gaming-focused UI variants
- Mobile optimization for smaller screens

## Risks & Concerns
- Not yet fully agentic puts Opera behind newer entrants
- 100M+ user base carries legacy tech debt
- Multi-model integration complicates support and quality control
- Gaming positioning is narrow niche vs mainstream browsers
