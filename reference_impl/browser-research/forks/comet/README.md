# Perplexity Comet

## Overview
Agentic AI browser built on Chromium fork, launched July 2025 with 1M+ downloads. Enables multi-site workflows combining research, comparison, and execution. Sidebar assistant for natural language task orchestration. Critical security/stability crisis in March 2026 caused App Store ranking collapse.

## Technical Approach
- **Base**: Chromium fork
- **AI Integration**: Perplexity AI sidebar assistant with autonomous action capability
- **Key Differentiator**: Multi-site workflow orchestration; research→comparison→execution in single session

## Key Learnings
- Agentic credential handling requires hardening before mass deployment
- March 2026 crisis reveals security/stability as critical blocker for agentic browsers
- User trust in autonomous actions is fragile—one incident can devastate adoption

## UX Innovations
- Sidebar assistant for persistent context across pages
- Natural language task execution across multiple domains
- Multi-site workflow coordination within single interface

## Risks & Concerns
- March 2026 security/stability crisis caused major regression
- Autonomous credential handling vulnerabilities
- User data protection concerns in agentic mode
- Recovery from reputation damage uncertain
