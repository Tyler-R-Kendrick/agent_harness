# Brave

## Overview
Privacy-first browser with hybrid AI approach. Leo assistant features zero server storage for conversations. Supports Bring-Your-Own-Model (BYOM). Agentic browsing capabilities released in Nightly (December 2025) with isolated profile and manual activation requirement.

## Technical Approach
- **Base**: Chromium
- **AI Integration**: Leo assistant with local-first storage; agentic mode in Nightly with isolated profiles
- **Key Differentiator**: Privacy-first architecture adds complexity but creates strong moat; experimental approach vs competitors' "ship hot"

## Key Learnings
- Privacy-first implementation significantly increases complexity but differentiates
- Isolated profiles for agentic mode reduces credential exposure risk
- Manual activation requirement signals cautious approach to autonomous actions
- BYOM support appeals to power users but complicates UX/support

## UX Innovations
- Zero-server-storage conversation model for privacy
- Isolated profile activation for safe agentic browsing
- BYOM support for advanced users
- Manual control over autonomous actions

## Risks & Concerns
- Complexity of privacy-first implementation could limit addressable market
- Nightly-only agentic features suggest not production-ready yet
- Manual activation adds friction vs competitors with always-on modes
- BYOM complicates support and error handling
