# Cromite

## Overview
Successor to Bromite project. Privacy and security-focused Chromium fork. Built-in ad blocking without extension overhead. Anti-fingerprinting features. Active development across Android and desktop platforms. Strong privacy posture as primary differentiation.

## Technical Approach
- **Base**: Chromium fork
- **AI Integration**: None (privacy-focused)
- **Key Differentiator**: Built-in ad blocking; anti-fingerprinting; no third-party extensions required

## Key Learnings
- Built-in ad blocking eliminates extension attack surface
- Anti-fingerprinting is valuable privacy feature
- Cross-platform (Android + desktop) increases addressable market
- Active Bromite successor shows project continuity matters

## UX Innovations
- Native ad blocking (no extension overhead)
- Anti-fingerprinting defenses
- Privacy-preserving defaults
- Cross-platform support

## Risks & Concerns
- Chromium update cadence requires continuous maintenance
- Built-in ad blocking narrows ecosystem (no ad blocking extensions)
- Cross-platform support increases maintenance burden
- Competing against privacy browser incumbents (Brave, Firefox)
