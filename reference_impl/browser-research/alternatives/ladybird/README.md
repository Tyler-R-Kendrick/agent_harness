# Ladybird

## Overview
Independent browser engine not derived from Chromium, WebKit, or Gecko. Written in C++ with transition to Rust underway. Alpha release planned for Summer 2026. Stable version targeted for 2028. Represents long-term effort to create engine independent from Chromium monoculture.

## Technical Approach
- **Base**: Custom engine (C++→Rust transition)
- **AI Integration**: None (pre-release)
- **Key Differentiator**: Independent from Chromium/WebKit/Gecko monoculture; long-term sustainability vision

## Key Learnings
- Independent engine development requires multi-year commitment
- C++→Rust transition is significant effort but improves long-term safety
- Summer 2026 alpha and 2028 stable targets are realistic timelines
- Chromium monoculture risk is creating motivation for alternatives

## UX Innovations
- Clean architectural foundation without legacy constraints
- Potential for novel interaction models
- Rust transition for memory safety

## Risks & Concerns
- Very long timeline (alpha 2026, stable 2028)
- Small team relative to browser scale
- Web compatibility is massive undertaking
- Business model/funding sustainability unclear
