## Workspace shell

Playwright test: `captures the main workspace screen`

![Workspace shell screenshot](./screenshots/workspace-screen.png)

How to interact:
- Use the omnibar to search or navigate.
- Switch workspaces with the pills under the omnibar.
- Browse tabs and folders from the workspace tree.
- Use the activity bar to switch between major panels.

## Settings / model registry

Playwright test: `captures the settings screen`

![Settings screen screenshot](./screenshots/settings-screen.png)

How to interact:
- Open **Settings** from the activity bar.
- Search the Hugging Face registry for browser-runnable ONNX models.
- Filter registry results by supported local task.
- Install a model for local browser inference.

## Extensions panel

Playwright test: `captures the extensions screen`

![Extensions screen screenshot](./screenshots/extensions-screen.png)

How to interact:
- Open **Extensions** from the activity bar.
- Review installed extension cards.
- Toggle extension enablement using the switch on each card.
