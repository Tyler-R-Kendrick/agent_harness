# Projects With Shared Memory

- Harness: ChatGPT
- Sourced: 2026-04-24

## What it is
Projects provide a persistent context hub with files, instructions, chats, and memory, and can be shared with collaborators.

## Evidence
- Help article: [Projects in ChatGPT](https://help.openai.com/en/articles/10169521)
- Built-in tools inside projects include:
  - Canvas
  - image generation
  - study mode
  - voice mode
  - web search
- Shared projects can expose chats, uploaded files, and custom instructions to collaborators.

## Product signal
Project-scoped memory is becoming a primary structure for long-running work in ChatGPT.
