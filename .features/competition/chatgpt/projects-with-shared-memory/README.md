# Projects With Shared Memory

- Harness: ChatGPT
- Sourced: 2026-05-21

## What it is
Projects are persistent ChatGPT workspaces with scoped memory, reusable source material, chat history, files, instructions, and collaboration boundaries that can be kept project-only when isolation matters.

## Evidence
- Help article: [Projects in ChatGPT](https://help.openai.com/en/articles/10169521)
- Built-in tools inside projects include:
  - Canvas
  - image generation
  - study mode
  - voice mode
  - web search
- Project behaviors called out:
  - project-only memory can isolate context from non-project chats
  - shared projects automatically become project-only memory
  - chats can reference other conversations inside the same project
  - sources can be added from Slack channels, Google Drive files/folders, saved chats, and ad-hoc pasted text
  - useful responses can be saved back into the project as reusable sources
- Shared projects can expose chats, uploaded files, and custom instructions to collaborators while still blocking access to members' non-project memories.
- Official visuals:
  - the help article includes the New Project dialog with Memory set to Project-only

## Product signal
OpenAI is turning projects into governed knowledge bases, not just folders for chats, with explicit source-ingestion paths and memory-boundary controls.
