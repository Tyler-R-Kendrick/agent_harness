# Scheduled Tasks

- Harness: ChatGPT
- Sourced: 2026-04-24

## What it is
Tasks let ChatGPT create scheduled work for itself and proactively deliver results later.

## Evidence
- Help article: [Tasks in ChatGPT](https://help.openai.com/en/articles/10291617)
- Supported behavior:
  - one-off or recurring schedules
  - API-triggered runs
  - execution while the user is offline
  - push notifications or email on completion

## Product signal
The recurring-work primitive is now first-class inside the consumer-facing product.
