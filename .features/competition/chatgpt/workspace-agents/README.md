# Workspace Agents

- Harness: ChatGPT
- Sourced: 2026-06-03

## What it is
ChatGPT workspace agents are shared, governable agents for repeatable business workflows across ChatGPT and Slack, with preview-before-publish, schedules, version history, and admin-controlled access.

## Evidence
- OpenAI product post: [Introducing workspace agents in ChatGPT](https://openai.com/index/introducing-workspace-agents-in-chatgpt/)
- Help article: [ChatGPT Workspace Agents for Enterprise and Business](https://help.openai.com/en/articles/20001143-chatgpt-workspace-agents-for-enterprise-and-business)
- Business release notes: [ChatGPT Business - Release Notes](https://help.openai.com/en/articles/11391654-chatgpt-business-release-notes)
- Capabilities called out:
  - shared agents in ChatGPT and Slack
  - schedules
  - apps and tools
  - team sharing and governance
  - create from templates or build from scratch
  - preview before publishing
  - add skills, files, and custom MCP servers
  - version history and analytics for agents
  - admin controls for agent building, publishing, and Slack usage

## Product signal
ChatGPT is moving from personal assistant UX toward organization-shared workflow agents.
