# Memory Sources And Personalization Controls

- Harness: ChatGPT
- Sourced: 2026-05-21

## What it is
Memory Sources expose which memories and prior context shaped a personalized response, and let the user directly correct, delete, or down-rank those sources without leaving the chat flow.

## Evidence
- Release notes: [ChatGPT release notes](https://help.openai.com/en/articles/6825453-chatgpt-release-notes)
- Help article: [Memory FAQ](https://help.openai.com/en/articles/8590148-memory-faq)
- The current product documentation calls out:
  - source chips for past chats, saved memories, and custom instructions across consumer plans
  - files in library and connected Gmail sources for eligible Plus/Pro users
  - source-level controls to mark context as relevant or not relevant
  - direct editing of saved memories and deletion of referenced chats
  - temporary chats and memory-off settings when personalization should not apply
- Release note date:
  - May 5, 2026: Memory sources and more personalized responses in ChatGPT

## Product signal
OpenAI is making personalization inspectable and steerable at the source level instead of treating memory as a hidden retrieval layer.
