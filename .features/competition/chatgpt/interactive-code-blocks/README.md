# Interactive Code Blocks

- Harness: ChatGPT
- Sourced: 2026-05-21

## What it is
Interactive Code Blocks let ChatGPT act more like an embedded artifact editor than a plain transcript, with inline editing, visual preview, and side-by-side review inside the conversation.

## Evidence
- Release notes: [ChatGPT release notes](https://help.openai.com/en/articles/6825453-chatgpt-release-notes)
- The February 19, 2026 release notes call out:
  - write and edit text in-line
  - preview diagrams and mini apps directly in chat
  - review code in split-screen views
- The February 27, 2026 web updates add:
  - export visuals from Code Blocks as images for reuse

## Product signal
OpenAI is moving generated code and visual outputs into editable in-chat artifacts, which shortens the gap between generation, review, and lightweight execution.
