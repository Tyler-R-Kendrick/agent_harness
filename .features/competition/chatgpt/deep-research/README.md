# Deep Research

- Harness: ChatGPT
- Sourced: 2026-05-21

## What it is
Deep research is a governed report workflow where ChatGPT proposes a plan, lets the user edit scope and trusted sources, runs while showing progress, and returns a documented report with traceable citations.

## Evidence
- Help article: [Deep research in ChatGPT](https://help.openai.com/en/articles/10500283-search-in-chatgpt)
- Release notes: [ChatGPT release notes](https://help.openai.com/en/articles/6825453-chatgpt-release-notes)
- Behavior called out:
  - user reviews/modifies a proposed plan
  - can use files, web, specified sites, and enabled apps
  - can be interrupted mid-run to refine focus
  - can adjust source access while running
  - returns a structured report with citations or source links
  - opens in a fullscreen report view with table of contents, sources used, and activity history
  - can be downloaded as Markdown, Word, or PDF

## Product signal
OpenAI is productizing plan-before-run oversight, trusted-source configuration, and review-grade deliverables as first-class agent behavior rather than leaving research inside opaque chat turns.
