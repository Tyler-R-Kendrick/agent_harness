# Deep Research

- Harness: ChatGPT
- Sourced: 2026-04-24

## What it is
Deep research turns multi-step information gathering into a structured, cited report workflow with source controls.

## Evidence
- Help article: [Deep research in ChatGPT](https://help.openai.com/articles/10500283)
- Behavior called out:
  - user reviews/modifies a proposed plan
  - can use files, web, specified sites, and enabled apps
  - returns a structured report with citations or source links

## Product signal
