# Apps And Connected Tools

- Harness: ChatGPT
- Sourced: 2026-04-24

## What it is
ChatGPT apps bring external tools and data into a conversation for search, action, deep research, and sync.

## Evidence
- Help article: [Apps in ChatGPT](https://help.openai.com/en/articles/11487775-connectors-in-chatgpt.webp)
- Capabilities called out:
  - in-chat actions
  - file/data search
  - deep research across connected sources
  - synced content for faster answers

## Product signal
Tool connectivity is being normalized as part of the core ChatGPT product, not a separate enterprise add-on.
