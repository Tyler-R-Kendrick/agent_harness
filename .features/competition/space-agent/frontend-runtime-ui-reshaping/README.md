# Frontend Runtime UI Reshaping

- Harness: Space Agent
- Sourced: 2026-04-30

## What it is
Space Agent positions the browser workspace itself as the agent runtime, so the agent can build pages, tools, widgets, and workflows directly into the live interface instead of staying trapped inside a fixed chat shell.

## Evidence
- Official README: [agent0ai/space-agent](https://github.com/agent0ai/space-agent)
- Official product site: [space-agent.ai login](https://space-agent.ai/login)
- First-party details:
  - the README says "the agent reshapes the interface"
  - the same page says users can ask for a page, tool, widget, or workflow and the agent can build it straight into the running workspace
  - the README also says the agent lives in the frontend runtime, specifically the browser layer itself
  - the hosted login page describes Space Agent as a free and open-source AI agent that "builds your space" right in the browser
- Latest development checkpoint:
  - the current public pitch still centers on runtime UI mutation rather than only prompt quality, which suggests this is the core product bet rather than a side demo

## Product signal
Space Agent treats the harness UI as agent-editable product surface, which is strategically different from assistants that can only answer within a prebuilt panel.
