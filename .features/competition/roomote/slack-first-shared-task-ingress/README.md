# Slack-First Shared Task Ingress

- Harness: Roomote
- Sourced: 2026-05-08

## What it is
Roomote positions Slack as the primary task surface for a coding agent that the whole company can use, while still accepting assignments from Linear and GitHub mentions.

## Evidence
- Official product page: [Roomote](https://roomote.dev/)
- First-party details:
  - Roomote says it "joins you in Slack and helps right away"
  - Slack is described as a "first-class entry surface"
  - teams can assign work from Linear
  - teams can get it fixing things from GitHub mentions
- Latest development checkpoint:
  - the current Roomote landing page presents Slack-native intake plus tracker and GitHub ingress as the default interaction model rather than as a later integration add-on

## Product signal
Roomote is optimizing for shared task intake inside the places work already starts, not for a single-user IDE-first workflow.
