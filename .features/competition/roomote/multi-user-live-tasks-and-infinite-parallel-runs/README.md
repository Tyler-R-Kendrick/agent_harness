# Multi-User Live Tasks And Infinite Parallel Runs

- Harness: Roomote
- Sourced: 2026-05-08

## What it is
Roomote supports multiple teammates participating in the same live task while also running many separate tasks in parallel.

## Evidence
- Official product page: [Roomote](https://roomote.dev/)
- First-party details:
  - Roomote says different teammates can run separate tasks in parallel
  - it also says 2+ people can chat with it in the same live multi-user task
  - the page describes "Infinite parallel chats"
  - Roomote says it can multiply itself in the cloud and that they usually run several per person at the same time
- Latest development checkpoint:
  - the product is combining parallel execution with collaborative steering in the same task surface, instead of forcing a choice between solo runs and shared review

## Product signal
Roomote is treating concurrency as both a scaling primitive and a collaboration primitive.
