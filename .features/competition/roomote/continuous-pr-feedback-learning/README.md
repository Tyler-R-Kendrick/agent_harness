# Continuous PR Feedback Learning

- Harness: Roomote
- Sourced: 2026-05-08

## What it is
Roomote says it keeps getting better by learning from pull-request feedback and refining its behavior over time.

## Evidence
- Official product page: [Roomote](https://roomote.dev/)
- First-party details:
  - Roomote says it "self-improves from PR feedback"
  - it says it keeps refining in the background "with your approval"
  - the product frames this as continuous improvement, not a one-off retry loop on a single task
- Latest development checkpoint:
  - Roomote is one of the clearest current examples of a commercial harness claiming explicit PR-feedback learning as a product feature, not just as internal model training lore

## Product signal
Roomote is moving toward a closed-loop system where review outcomes become reusable operating knowledge for later runs.
