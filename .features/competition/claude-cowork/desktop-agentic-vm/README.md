# Desktop Agentic VM

- Harness: Claude Cowork
- Sourced: 2026-04-24

## What it is
Cowork brings Claude Code's agentic architecture into Claude Desktop and runs tasks in an isolated VM on the user's computer.

## Evidence
- Help article: [Get started with Claude Cowork](https://support.claude.com/en/articles/13345190-get-started-with-cowork)
- Claimed behavior:
  - local file access through the desktop app
  - isolated VM execution
  - multi-step task planning
  - finished outputs written back to the file system

## Product signal
Anthropic is packaging a local execution environment for non-terminal users.
