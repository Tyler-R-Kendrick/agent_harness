# Projects With Local Memory And Instructions

- Harness: Claude Cowork
- Refreshed: 2026-05-23

## What it is
Cowork projects are local, task-scoped workspaces that package instructions, context, files, memory, and scheduled work around a recurring stream of knowledge-work tasks.

## Evidence
- Official docs: [Organize your tasks with projects in Claude Cowork](https://support.claude.com/en/articles/14116274-organize-your-tasks-with-projects-in-cowork)
- First-party details:
  - projects group related tasks into dedicated workspaces with their own files, context, instructions, and memory
  - users can create a project from scratch, import from an existing Claude chat project, or wrap an existing local folder
  - each project carries project-specific instructions, scheduled tasks, context, and project-scoped memory
  - project memory stays isolated per project instead of leaking across unrelated work
  - projects are local to the desktop app and do not currently sync through the cloud
- Latest development checkpoint:
  - the April 9, 2026 projects launch makes Cowork's workspace model much more explicit than the older one-task-at-a-time preview

## Product signal
Anthropic is turning memory and context from a loose account-level feature into a durable local workspace contract, which is closer to how serious harness users actually organize repeated operational work.
