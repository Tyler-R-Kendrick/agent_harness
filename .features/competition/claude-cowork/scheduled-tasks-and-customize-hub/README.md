# Scheduled Tasks And Customize Hub

- Harness: Claude Cowork
- Refreshed: 2026-05-23

## What it is
Cowork can run recurring or on-demand tasks automatically and centralizes skills, plugins, and connectors under a shared Customize surface inside Claude Desktop.

## Evidence
- Release notes: [February 25, 2026 Cowork update](https://support.claude.com/en/articles/12138966-release-notes)
- Official docs: [Schedule recurring tasks in Claude Cowork](https://support.claude.com/en/articles/13854387-schedule-recurring-tasks-in-claude-cowork)
- First-party details:
  - scheduled tasks can run automatically on a cadence or be saved for on-demand reuse
  - scheduled tasks reuse the same connected tools, plugins, and skills available to normal Cowork tasks
  - the Customize section groups skills, plugins, and connectors in one place instead of scattering setup across the UI
  - scheduled tasks are visible and manageable from a dedicated Scheduled section in the sidebar
  - runs happen only while the user's computer is awake and Claude Desktop remains open
- Latest development checkpoint:
  - Anthropic now treats recurring automation and workflow packaging as one joined feature area, not separate advanced-user features

## Product signal
Cowork is converging toward a packaging-plus-automation model where users define durable capabilities once, then schedule them instead of repeatedly re-prompting.
