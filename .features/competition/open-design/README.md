# Open Design

- Harness: Open Design
- Sourced: 2026-05-13

## What it is
Open Design is a local-first design-agent harness that wraps existing coding-agent CLIs in a schema-driven skill system, deterministic design-system context, sandboxed artifact previews, and deployable design outputs.

## Researched feature set
- `structured-brief-locking-and-direction-picker`
- `design-systems-and-craft-rule-injection`
- `cross-agent-skill-protocol-with-typed-inputs-and-live-sliders`
- `sandboxed-artifact-preview-and-multi-format-export`
- `critique-theater-and-agent-emitted-tweak-controls`
- `multimodal-media-generation-and-hyperframes`
- `multi-cli-agent-runtime-and-byok-proxy`
- `external-mcp-client-and-daemon-managed-oauth`
- `cloudflare-pages-deployment-and-custom-domains`
- `imports-and-persistent-local-project-state`

## Screenshots and demos
- Official README demo gallery: entry view, discovery form, direction picker, live todo progress, sandboxed preview, design-system library, deck mode, and mobile prototype screenshots are published in the project README.

## Product signal
Open Design is pushing a strong pattern where skills are not just prompts or tools; they are typed contracts that can generate the intake UI, preview behavior, export pipeline, and refinement controls around the agent.
