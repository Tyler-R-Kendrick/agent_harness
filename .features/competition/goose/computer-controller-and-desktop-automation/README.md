# Computer Controller And Desktop Automation

- Harness: Goose
- Sourced: 2026-05-03

## What it is
Goose includes a built-in Computer Controller extension for web interaction and desktop automation, and recent work rebuilt that surface around annotated screenshots and direct GUI actions.

## Evidence
- Official docs: [Computer Controller Extension](https://goose-docs.ai/docs/mcp/computer-controller-mcp)
- Official blog: [Beyond the Terminal: goose Controls Your Desktop with Peekaboo](https://goose-docs.ai/blog/2026/04/29/computer-controller-peekaboo)
- First-party details:
  - Goose documents the Computer Controller extension as a built-in way to automate web interaction, application control, and file-oriented desktop tasks.
  - The extension includes web search and scraping alongside general computer-control abilities.
  - The April 29, 2026 blog says the extension was rebuilt with Peekaboo so Goose can inspect annotated screenshots, identify UI elements, click, type, scroll, drag, and navigate menus across apps on macOS.
  - The same post frames the feature as Goose breaking out of the terminal into direct GUI control.
- Latest development checkpoint:
  - Goose's late-April 2026 desktop-control push is one of the clearest recent examples of an agent harness extending from coding into computer use

## Product signal
Goose is converging terminal workflows and direct GUI execution. That makes it a closer competitor for browser and desktop agents than a typical code-only assistant.
