# n8n

- Harness: n8n
- Sourced: 2026-05-10

## What it is
n8n is an automation-first agent harness that treats agents as nodes inside versioned workflows instead of as a single chat surface.

## Researched feature set
- `ai-workflow-builder`
- `mcp-server-builds-and-updates-workflows`
- `chat-hub-and-workflow-agents`
- `multi-agent-delegation-via-agent-tool`
- `workflows-as-tools-with-ai-parameter-binding`
- `human-review-for-tool-calls`
- `chat-pause-resume-human-checkpoints`
- `evaluation-native-agent-workflows`
- `versioned-publishing-and-concurrency-protection`

## Product signal
n8n is a strong competitor wherever the differentiator is not just agent quality, but the ability to wrap agents in deterministic workflow logic, approvals, evaluations, and deployment-safe operations.
