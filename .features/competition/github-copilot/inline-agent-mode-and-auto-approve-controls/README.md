# Inline Agent Mode And Auto-Approve Controls

- Harness: GitHub Copilot
- Sourced: 2026-05-05

## What it is
Copilot is pushing agent mode into inline editor flows while also exposing stronger approval controls for terminal commands, file edits, and other tool calls.

## Evidence
- Docs: [GitHub Copilot features](https://docs.github.com/en/copilot/get-started/features)
- Changelog: [Inline agent mode in preview and more in GitHub Copilot for JetBrains IDEs](https://github.blog/changelog/2026-04-24-inline-agent-mode-in-preview-and-more-in-github-copilot-for-jetbrains-ides/)
- GitHub documents:
  - agent mode in IDEs where Copilot chooses files, proposes terminal commands, and iterates toward completion
  - inline agent mode in JetBrains so agent behavior can run from inline chat instead of only the side panel
  - global auto-approve and granular defaults for unmatched commands and file edits

## Product signal
GitHub is collapsing the gap between inline assistance and full agent execution while making approval policy more configurable.
