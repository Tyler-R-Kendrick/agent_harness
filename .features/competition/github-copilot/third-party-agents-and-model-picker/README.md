# Third-Party Agents And Model Picker

- Harness: GitHub Copilot
- Sourced: 2026-05-05

## What it is
GitHub Copilot now lets users run third-party coding agents such as Claude and Codex inside the same GitHub task surface, with shared governance and per-agent model selection.

## Evidence
- Docs: [GitHub Copilot features](https://docs.github.com/en/copilot/get-started/features)
- Docs: [About third-party agents](https://docs.github.com/en/copilot/concepts/agents/about-third-party-agents)
- Changelog: [Claude and Codex are now available in public preview on GitHub](https://github.blog/changelog/2026-02-04-claude-and-codex-are-now-available-in-public-preview-on-github/)
- Changelog: [Model selection for Claude and Codex agents on github.com](https://github.blog/changelog/2026-04-14-model-selection-for-claude-and-codex-agents-on-github-com/)
- GitHub documents:
  - Claude and Codex can be launched from the agents tab, issues, pull requests, GitHub Mobile, and VS Code
  - third-party agents inherit the same security protections and policy controls as Copilot cloud agent
  - GitHub exposes per-agent model selection on github.com

## Product signal
GitHub is becoming a policy-governed control plane for multiple coding agents instead of a single bundled assistant.
