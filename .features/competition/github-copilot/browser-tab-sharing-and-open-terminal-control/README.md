# Browser Tab Sharing And Open Terminal Control

- Harness: GitHub Copilot
- Sourced: 2026-05-18

## What it is
Copilot agents can now use shared live browser tabs as execution context and can read from or write to already-open terminals, including REPLs and interactive scripts.

## Evidence
- Changelog: [GitHub Copilot in Visual Studio Code, April releases](https://github.blog/changelog/2026-05-06-github-copilot-in-visual-studio-code-april-releases/)
- Changelog: [GitHub Copilot in Visual Studio Code, March Releases](https://github.blog/changelog/2026-04-08-github-copilot-in-visual-studio-code-march-releases/)
- GitHub documents:
  - agents gain an integrated browser with on-demand tab sharing
  - agents can read page content, interact with pages, and validate changes in real time
  - agents can read from and write to existing foreground terminals instead of only spawning isolated commands
  - terminal access includes running REPLs and other interactive scripts

## Product signal
GitHub is collapsing the gap between passive chat context and live runtime control across browser and terminal surfaces.
