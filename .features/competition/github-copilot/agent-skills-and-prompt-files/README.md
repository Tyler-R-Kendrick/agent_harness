# Agent Skills And Prompt Files

- Harness: GitHub Copilot
- Sourced: 2026-05-05

## What it is
Copilot supports reusable prompt files, Agent Skills, and generated customizations so teams can package or draft repeatable instructions, scripts, and resources directly in the repo or editor.

## Evidence
- Docs: [GitHub Copilot features](https://docs.github.com/en/copilot/get-started/features)
- Changelog: [GitHub Copilot now supports Agent Skills](https://github.blog/changelog/2025-12-18-github-copilot-now-supports-agent-skills/)
- Changelog: [GitHub Copilot in Visual Studio Code, April releases](https://github.blog/changelog/2026-05-06-github-copilot-in-visual-studio-code-april-releases/)
- Changelog: [GitHub Copilot in Visual Studio — April update](https://github.blog/changelog/2026-04-30-github-copilot-in-visual-studio-april-update/)
- GitHub documents:
  - prompt files as workspace Markdown prompts with reusable context
  - Agent Skills as folders of instructions, scripts, and resources
  - skills working across cloud agent, Copilot CLI, and IDE agent mode
  - compatibility with existing `.claude/skills` repositories
  - natural-language generation of custom agents, skills, and instructions from inside VS Code
  - Visual Studio support for skill discovery in `.claude/skills/` and `.agents/skills/` in addition to `.github/skills/`

## Product signal
GitHub is converging on repo-native workflow packaging instead of keeping advanced agent behavior trapped in per-user prompts.
