## Notes
- This refresh extends the existing Copilot corpus with the May 2026 GitHub Copilot app surface and the newer Copilot Memory governance controls.
- The biggest new signals are a GitHub-native desktop workbench for parallel branch-isolated agent sessions, scheduled workflows inside that app, and repo or enterprise memory controls with explicit review and delete flows.
- First-party visuals are available in the Copilot app and Copilot Memory docs, including screenshots of app workflows and repository or user memory settings.
