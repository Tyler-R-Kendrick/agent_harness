# GUI Slash Menu And Skill Discovery

- Harness: OpenHands
- Sourced: 2026-05-02

## What it is
OpenHands added a GUI slash menu so users can discover and invoke loaded Agent Skills directly from the chat composer rather than remembering names or re-describing a workflow from scratch.

## Evidence
- Official changelog: [OpenHands Product Update - March 2026](https://openhands.dev/blog/openhands-product-update---march-2026)
- First-party details:
  - typing `/` in the GUI opens a list of Agent Skills loaded in the current conversation context
  - the menu is meant to make loaded skills easier to find and apply quickly
  - OpenHands says future enhancements will extend this toward native slash commands for plugins and other workflows
- Latest development checkpoint:
  - the March 6, 2026 product update introduced the GUI Slash Menu and included product imagery on the source page

## Product signal
This is a strong discoverability move. Skill systems become much more useful once the product helps users find the right workflow package at the point of task entry.
