# Multi-Agent Command Center

- Harness: Codex
- Sourced: 2026-04-24

## What it is
Codex app is positioned as a command center for supervising multiple agents, each running in separate threads and isolated worktrees.

## Evidence
- OpenAI product post: [Introducing the Codex app](https://openai.com/index/introducing-the-codex-app/)
- Key details:
  - agents can be managed "at once" and run "in parallel"
  - work is organized by project and thread
  - built-in worktree support isolates each agent's copy of the repo

## Product signal
This is a strong move from single-agent chat UX toward explicit orchestration and conflict-free parallel execution.
