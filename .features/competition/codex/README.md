## What it is
Codex is a coding-agent harness that now spans local, isolated-worktree, and cloud execution modes across desktop, terminal, IDE, GitHub, Linear, browser, computer-use, remote-mobile control, explicit goals, window-context capture, and workspace-governed automation surfaces.

## Product signal
Codex is no longer just a local coding assistant. The strongest current signal is execution portability plus client portability: the same task can start locally, move into a worktree or cloud run, pick up goal state, absorb live window context, expose streamed events through app-server, surface in GitHub or Linear, and still be steered from the desktop app, IDE, terminal, or phone.
