## What it is
Codex is a coding-agent harness that now spans local, isolated-worktree, and cloud execution modes across desktop, terminal, IDE, GitHub, Linear, browser, computer-use, and remote-mobile control surfaces.

## Product signal
Codex is no longer just a local coding assistant. The strongest current signal is execution portability: the same task can start locally, move into a worktree or cloud run, surface in GitHub or Linear, and still be steered from the desktop app, IDE, or phone.
