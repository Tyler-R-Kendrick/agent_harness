# Configurable Personality

- Harness: Codex
- Sourced: 2026-05-16

## What it is
Codex supports multiple interaction styles without changing core capabilities.

## Evidence
- Official product post: [Introducing the Codex app](https://openai.com/index/introducing-the-codex-app/)
- First-party details:
  - the app, CLI, and IDE extension expose a `/personality` command
  - OpenAI explicitly calls out a terse and pragmatic mode plus a more conversational and empathetic mode
  - OpenAI frames the difference as style selection without changing Codex capabilities
- Latest development checkpoint:
  - current first-party messaging makes personality a portable cross-surface setting instead of a hidden system-prompt choice

## Product signal
Personality is being surfaced as a user-facing workflow control, not just a hidden prompt choice.
