# Configurable Personality

- Harness: Codex
- Sourced: 2026-04-24

## What it is
Codex supports multiple interaction styles without changing core capabilities.

## Evidence
- Source: [Introducing the Codex app](https://openai.com/index/introducing-the-codex-app/)
- The app, CLI, and IDE extension expose a `/personality` command.
- OpenAI explicitly calls out a terse/pragmatic option and a more conversational/empathetic option.

## Product signal
Personality is being surfaced as a user-facing workflow control, not just a hidden prompt choice.
