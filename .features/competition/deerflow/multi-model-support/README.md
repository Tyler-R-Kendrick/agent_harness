# Multi-Model Support

- Harness: DeerFlow
- Sourced: 2026-04-26

## What it is
DeerFlow is model-agnostic and can route across hosted APIs, OpenAI-compatible gateways, CLI-backed providers, and local inference stacks.

## Evidence
- Official site: [DeerFlow](https://deerflow.tech/)
- GitHub README: [DeerFlow - 2.0](https://github.com/bytedance/deer-flow/blob/main/README.md)
- First-party details:
  - the site advertises support for Doubao, DeepSeek, OpenAI, Gemini, and others
  - the README provides configuration examples for OpenAI, OpenRouter, Responses API, vLLM, Codex CLI, and Claude Code OAuth
  - recommended model guidance focuses on long context, reasoning, multimodal inputs, and tool use rather than a single vendor stack

## Product signal
DeerFlow is positioning the harness as portable execution infrastructure that survives model churn.
