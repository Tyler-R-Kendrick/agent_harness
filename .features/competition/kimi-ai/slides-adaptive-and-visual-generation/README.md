# Slides Adaptive And Visual Generation

- Harness: Kimi AI
- Sourced: 2026-05-11

## What it is
Kimi Slides is a deliverable-specific presentation agent with two distinct generation modes: a slower adaptive mode for custom, research-backed design and a faster template-driven visual mode for high-speed deck production.

## Evidence
- Official docs: [Kimi Slides](https://www.kimi.com/help/slides/ppt-overview)
- Official docs: [Difference between adaptive and visual](https://www.kimi.com/help/slides/ppt-creation-mode)
- First-party details:
  - Kimi Slides can generate from a one-line topic, pasted text, or uploaded documents and supports PDF, Word, PPTX, Excel, TXT, and images as inputs
  - Kimi says the system automatically researches, writes, structures, and designs the deck
  - Adaptive mode generates layout and styling automatically, supports multiple style presets, and typically takes 30 to 60 minutes
  - Visual mode uses prebuilt templates, is powered by Nano Banana Pro, and usually takes 5 to 10 minutes
  - users can upload a PPTX template and ask Kimi to generate against that custom visual system
- Latest development checkpoint:
  - the current Slides docs make the dual-mode model explicit, separating unique-design generation from fast template filling as two different operational paths inside one presentation harness

## Product signal
Kimi is not just generating decks; it is productizing a time-vs-customization tradeoff as an explicit control surface for agentic presentation work.
