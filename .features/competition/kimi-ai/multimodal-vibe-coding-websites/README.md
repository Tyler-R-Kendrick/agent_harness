# Multimodal Vibe Coding Websites

- Harness: Kimi AI
- Sourced: 2026-05-11

## What it is
Kimi Websites is a multimodal website-building agent that turns natural language, screenshots, mockups, or screen recordings into runnable sites, then iterates through preview, annotation, and deployment loops.

## Evidence
- Official docs: [Kimi Websites overview](https://www.kimi.com/help/websites/Websites-overview)
- First-party details:
  - Kimi says Websites is powered by its latest K2.6 multimodal visual agent model
  - the feature supports conversational site building, image and video understanding, auto-deploy, multi-round editing, one-click sharing, template selection, and code export
  - Kimi describes a dedicated website-building skill that analyzes requirements, plans sub-tasks, chooses a stack such as React plus Tailwind CSS, generates assets, writes code, and self-adjusts from preview results
  - the editing loop supports live preview plus visual annotation and selection modes so users can mark up exact areas for revision
  - Kimi positions the surface for developers, product managers, entrepreneurs, and non-technical users
- Latest development checkpoint:
  - the current Websites docs frame this as `Vibe Coding` and emphasize that the agent can keep reorganizing its internal to-do list as the user changes direction mid-build

## Product signal
Kimi is packaging website generation as a specialized agent runtime with its own iterative editing ergonomics instead of making users drive the whole workflow from a generic chat thread.
