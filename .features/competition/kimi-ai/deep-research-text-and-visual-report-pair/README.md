# Deep Research Text And Visual Report Pair

- Harness: Kimi AI
- Sourced: 2026-05-11

## What it is
Deep Research is Kimi's asynchronous research harness that asks clarifying questions, runs broad search and reasoning loops, and returns both a long-form written report and a shareable visual artifact.

## Evidence
- Official docs: [Deep Research](https://www.kimi.com/help/deep-research/deep-research-overview)
- First-party details:
  - Kimi calls Deep Research its first Agent product
  - the pipeline includes clarification, deep reasoning, active search, iterative reasoning, tool invocation, and final report synthesis
  - Kimi says a typical task averages 23 reasoning steps, plans about 74 keywords, discovers 206 URLs, and filters to the top 3.2% of sources
  - each run is asynchronous and usually takes 10 to 25 minutes
  - the output includes a 10,000+ word text report with citations plus a visual HTML report with structured layouts, mind maps, and a public sharing link
- Latest development checkpoint:
  - Kimi's current Deep Research help center now frames the visual report as a first-class second deliverable rather than just an optional viewer around the text result

## Product signal
Kimi is treating research deliverables as dual artifacts, with a citation-heavy document for rigor and a shareable visual summary for distribution and review.
