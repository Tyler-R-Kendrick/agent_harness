# Claw Groups Shared Workspace For Heterogeneous Agents

- Harness: Kimi AI
- Sourced: 2026-05-27

## What it is
Claw Groups is Kimi's preview surface for coordinating multiple humans and agents in one shared workspace, even when those agents use different tools, contexts, and models.

## Evidence
- Official model page: [Kimi K2.6](https://www.kimi.com/ai-models/kimi-k2-6)
- Official help: [Kimi Claw overview](https://www.kimi.com/help/kimi-claw/overview)
- First-party details:
  - Kimi says Claw Groups lets users bring agents with different tools, contexts, and models into a shared space
  - the coordinator assigns tasks, manages dependencies, and keeps the workflow on track
  - the K2.6 page frames this as a team surface where users set the goal and the coordinated group handles the rest
  - the Kimi Claw help center now exposes a dedicated `Claw Group Chat overview` entry, which shows the feature moving from blog-only language into the product navigation
- Latest development checkpoint:
  - the current K2.6 product surface upgrades Kimi from "many spawned sub-agents" to "shared workspaces for heterogeneous agents," which materially broadens the orchestration model beyond same-runtime specialist workers

## Product signal
Kimi is moving toward bring-your-own-agent teamspaces, where orchestration is not limited to one model family or one local runtime.
