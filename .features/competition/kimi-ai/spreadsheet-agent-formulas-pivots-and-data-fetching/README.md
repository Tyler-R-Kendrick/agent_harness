# Spreadsheet Agent Formulas Pivots And Data Fetching

- Harness: Kimi AI
- Sourced: 2026-05-11

## What it is
Kimi Sheets is an Excel-native spreadsheet agent that builds structured workbooks, writes formulas, links sheets, creates pivots and charts, and can fetch or clean data as part of the same flow.

## Evidence
- Official feature page: [Kimi Sheets](https://www.kimi.com/features/sheets)
- Official feature index: [Kimi Features](https://www.kimi.com/features/)
- First-party details:
  - Kimi says Sheets handles everything from simple data tables to multi-sheet financial analysis
  - the agent automatically generates formulas, organizes data into tables, and fills workbook content from plain-language requests
  - it supports conversion between Excel and formats such as PDF, Word, PPT, CSV, and JSON
  - Kimi highlights native formula generation, multi-sheet logic modeling, automated pivot tables, and chart generation
  - the product also claims smart data fetching and cleaning from sources like Yahoo Finance and ArXiv
- Latest development checkpoint:
  - the current Sheets page pushes beyond spreadsheet drafting into workbook-native analysis and modeling, especially linked-sheet logic and external data ingestion

## Product signal
Kimi is treating spreadsheets as a structured computational surface, not as a text prompt that happens to emit CSV-like output.
