# MCP And Custom Tools

- Harness: OpenCode
- Sourced: 2026-04-28

## What it is
OpenCode supports both built-in tools and extensibility through custom tools and MCP servers.

## Evidence
- Official docs: [Tools](https://opencode.ai/docs/tools/) and [MCP servers](https://opencode.ai/docs/mcp-servers)
- First-party details:
  - the tools docs say OpenCode can be extended with custom tools or MCP servers
  - MCP configuration supports both `local` and `remote` servers
  - MCP tools can be enabled globally or only for selected agents
  - the docs include examples for Context7 and Grep by Vercel

## Product signal
OpenCode is competing on action surface breadth, not just core prompting quality.
