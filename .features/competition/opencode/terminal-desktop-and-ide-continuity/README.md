# Terminal Desktop And IDE Continuity

- Harness: OpenCode
- Sourced: 2026-04-28

## What it is
OpenCode spans terminal, desktop, and IDE surfaces so the same harness can follow the user across environments.

## Evidence
- Official site: [OpenCode](https://opencode.ai/)
- Official docs: [Intro](https://opencode.ai/docs/) and [IDE](https://opencode.ai/docs/ide/)
- First-party details:
  - the product page says OpenCode is available in the terminal, desktop app, and IDE extension
  - the site currently advertises a desktop beta for macOS, Windows, and Linux
  - the IDE docs cover VS Code, Cursor, and other terminal-capable IDEs
  - the IDE flow supports quick launch, new session shortcuts, selection-aware context, and file reference shortcuts

## Product signal
The harness is becoming a persistent system layer rather than a single-interface tool.
