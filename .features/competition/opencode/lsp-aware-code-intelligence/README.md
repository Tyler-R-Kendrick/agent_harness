# LSP-Aware Code Intelligence

- Harness: OpenCode
- Sourced: 2026-04-28

## What it is
OpenCode integrates with LSP servers so the model can work with diagnostics and language-aware project context.

## Evidence
- Official site: [OpenCode](https://opencode.ai/)
- Official docs: [LSP Servers](https://opencode.ai/docs/lsp/)
- First-party details:
  - the product page lists `LSP enabled` as a headline capability
  - the docs say OpenCode integrates with LSP servers and uses diagnostics to provide feedback to the LLM
  - built-in support spans many languages and can auto-install selected servers
  - users can disable, override, or add custom LSP servers in config

Harnesses are increasingly using language-aware tooling to reduce low-context edits and improve repository fit.
