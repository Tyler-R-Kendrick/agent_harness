# Multi-Provider Model Access

- Harness: OpenCode
- Sourced: 2026-04-28

## What it is
OpenCode is model-neutral and supports a large provider matrix, local models, and configurable model variants.

## Evidence
- Official site: [OpenCode](https://opencode.ai/)
- Official docs: [Models](https://opencode.ai/docs/models/) and [Providers](https://opencode.ai/docs/providers/)
- First-party details:
  - the product page says users can connect "any model from any provider"
  - the docs say OpenCode supports `75+ LLM providers` through Models.dev and supports local models
  - models can be selected with `/models`
  - providers and per-model variants can be configured in `opencode.json`

## Product signal
Provider neutrality is becoming a core differentiator for harnesses that want to avoid single-vendor lock-in.
