# Multi-Provider Model Access

- Harness: OpenCode
- Sourced: 2026-04-27

## What it is
OpenCode is designed to work across many model providers, local models, and even existing paid accounts such as GitHub Copilot and ChatGPT Plus/Pro.

## Evidence
- Official product site: [OpenCode](https://dev.opencode.ai/)
- Official docs: [Providers](https://opencode.ai/docs/providers/)
- Official docs: [Models](https://opencode.ai/docs/models/)
- First-party details:
  - the homepage says OpenCode supports "75+ LLM providers" and local models
  - the homepage highlights GitHub Copilot login and ChatGPT Plus/Pro login as supported account paths
  - provider credentials can be added with `/connect`
  - model variants and provider-specific options can be configured in `opencode.json`

## Product signal
OpenCode is positioning model access breadth as a feature, lowering switching cost and letting users optimize for cost, latency, or quality without changing harnesses.
