# Terminal Desktop IDE Continuity

- Harness: OpenCode
- Sourced: 2026-04-27

## What it is
OpenCode is intentionally multi-surface, spanning terminal, desktop, and IDE use with context handoff and session launch shortcuts.

## Evidence
- Official product site: [OpenCode](https://dev.opencode.ai/)
- Official docs: [IDE](https://opencode.ai/docs/ide/)
- First-party details:
  - the homepage says OpenCode is available in the terminal, desktop app, and IDE extension
  - the homepage notes the desktop app beta is available on macOS, Windows, and Linux
  - the IDE docs describe quick launch and new-session shortcuts from VS Code and related editors
  - IDE context awareness can share the current selection or tab with OpenCode

## Product signal
OpenCode is competing as a continuous environment rather than a terminal-only coding agent.
