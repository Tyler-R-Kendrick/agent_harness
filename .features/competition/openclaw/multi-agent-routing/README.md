# Multi-Agent Routing

- Harness: OpenClaw
- Research date: 2026-04-23
- What it does: OpenClaw can route inbound channels, accounts, and peers to isolated agents with separate workspaces and per-agent sessions.
- Why it matters: Isolation per persona, channel, or task is a recurring pattern in newer harnesses that want to support multi-context work safely.
