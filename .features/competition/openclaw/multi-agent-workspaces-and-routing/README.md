# Multi-Agent Workspaces And Routing

- Harness: OpenClaw
- Sourced: 2026-04-24

## What it is
OpenClaw exposes isolated agents with separate workspaces and channel bindings.

## Evidence
- Docs page: [openclaw agents](https://github.com/openclaw/openclaw/blob/main/docs/cli/agents.md)
- The CLI supports:
  - `agents add`
  - workspace assignment
  - routing bindings
  - identity changes
  - per-agent bindings

## Product signal
OpenClaw has an explicit routing model for multi-agent operation across channels.
