# Chat-App-Everywhere Interface

- Harness: OpenClaw
- Sourced: 2026-04-24

## What it is
OpenClaw presents messaging platforms as its primary user interface.

## Evidence
- Official site: [Open Claw Agent](https://agentopenclaw.io/)
- The site calls out WhatsApp, Telegram, Discord, Slack, Signal, and iMessage as interfaces.

## Product signal
The harness is designed for ambient invocation from wherever the user already communicates.
