# MCP Apps And Team Plugin Marketplaces

- Harness: Cursor
- Sourced: 2026-05-01

## What it is
Cursor is turning MCP and plugins into a richer platform surface by rendering interactive app UIs inside chats and by giving teams governed internal marketplaces for private plugins.

## Evidence
- Official changelog: [MCP Apps and Team Marketplaces for Plugins](https://cursor.com/changelog/2-6)
- Official changelog: [New Plugins on the Cursor Marketplace](https://cursor.com/en/changelog)
- First-party details:
  - the March 3, 2026 `2.6` release says MCP Apps can render interactive interfaces such as charts, diagrams, and whiteboards directly inside Cursor
  - the same release adds team marketplaces so admins can distribute private plugins with central governance and access controls
  - Cursor also reports improved MCP setup, configuration, and tool-call UX in the same release, which matters because platform features only help if setup friction drops
  - the current changelog highlights new partner plugins across a wider stack, showing ongoing investment in the marketplace rather than a one-time platform launch
- Latest development checkpoint:
  - the March 2026 platform push makes Cursor's tool surface feel more like an internal app platform than a simple model-plus-terminal wrapper

## Product signal
Cursor is productizing tool integrations as shared platform assets, which increases lock-in and raises expectations for governance, distribution, and richer in-chat UI.
