# Background Workflows And Shortcuts

- Harness: Claude in Chrome
- Sourced: 2026-04-24

## What it is
Claude in Chrome supports background workflows, saved shortcuts, and scheduled shortcuts.

## Evidence
- Help article: [Get started with Claude in Chrome](https://support.claude.com/en/articles/12012173-get-started-with-claude-in-chrome)
- Documented capabilities:
  - continue working while user switches tabs
  - reusable slash-style shortcuts
  - scheduled shortcuts

## Product signal
This turns browser automation into repeatable operations instead of ad hoc prompting.
