# Multi-Tab Groups

- Harness: Claude in Chrome
- Sourced: 2026-04-24

## What it is
Claude can view and interact with multiple tabs at once by using a designated tab group.

## Evidence
- Help article: [Get started with Claude in Chrome](https://support.claude.com/en/articles/12012173-get-started-with-claude-in-chrome)
- The core capabilities section explicitly documents multi-tab functionality.

## Product signal
Anthropic is addressing one of the main real-world bottlenecks in browser automation: context across more than one tab.
