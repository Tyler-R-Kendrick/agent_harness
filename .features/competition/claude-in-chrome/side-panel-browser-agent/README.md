# Side Panel Browser Agent

- Harness: Claude in Chrome
- Sourced: 2026-04-24

## What it is
Claude in Chrome is a browser extension that reads, clicks, and navigates websites from a side panel.

## Evidence
- Help article: [Get started with Claude in Chrome](https://support.claude.com/en/articles/12012173-get-started-with-claude-in-chrome)
- Anthropic states that Claude works in the side panel while browsing and can act on websites.

## Product signal
This is a direct browser-native harness, not a generic chat tab with links.
