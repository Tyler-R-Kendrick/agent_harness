# Permission Modes

- Harness: Claude in Chrome
- Research date: 2026-04-23
- What it does: Claude in Chrome supports at least two explicit permission modes: ask before acting and act without asking.
- Why it matters: Permission posture needs to be obvious and easy to switch for browser agents exposed to prompt-injection risk.
