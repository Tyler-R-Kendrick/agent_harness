# Desktop Connector

- Harness: Claude in Chrome
- Research date: 2026-04-23
- What it does: The Chrome extension can be enabled as a connector inside Claude Desktop so desktop tasks can hand work off to the browser without manual switching.
- Why it matters: Cross-surface interoperability is becoming a strong differentiator for agent platforms.
