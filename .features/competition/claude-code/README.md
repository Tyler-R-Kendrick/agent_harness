# Claude Code

- Harness: Claude Code
- Last researched: 2026-05-22

## What it is
Claude Code is Anthropic's coding harness across terminal, IDE, browser, GitHub, and remote-control surfaces. The current docs show it evolving from a terminal copilot into a supervised multi-agent system with background sessions, policy hooks, MCP connectivity, and cloud-to-local continuity.

## Key feature areas
- `git-native-code-agent`: branch, diff, command, and verification flow as a first-class coding path
- `subagents`: specialized subagents with isolated context, prompts, and tool permissions
- `claude-md-auto-memory-hooks`: layered `CLAUDE.md`, auto-memory, import support, and programmable hooks
- `mcp-tool-connectivity`: MCP-based external tool and data connectivity
- `web-and-ide-continuity`: terminal, IDE, and web continuity for the same harness
- `agent-view-background-session-supervision`: background session list with peek, reply, and status visibility
- `worktree-backed-parallel-sessions`: isolated git worktrees for concurrent Claude Code tasks
- `github-actions-code-review-and-ci-delegation`: GitHub Actions review and automation paths for remote execution
- `remote-control-and-mobile-follow-through`: remote control from other surfaces into active local sessions
