# GitHub Actions Code Review And CI Delegation

- Harness: Claude Code
- Sourced: 2026-05-22

## What it is
Claude Code supports GitHub-native remote execution through GitHub Actions so review or automation tasks can run in CI instead of only on a developer's local machine.

## Evidence
- Docs: [GitHub Actions](https://code.claude.com/docs/en/github-actions)
- The docs cover using Claude Code in Actions for:
  - pull-request review and follow-up automation
  - issue or workflow triggered code tasks
  - remote execution in a controlled CI environment with repo-native results
