# Remote Control And Mobile Follow-Through

- Harness: Claude Code
- Sourced: 2026-05-22

## What it is
Claude Code supports remote control flows so users can monitor or continue local sessions from other devices and surfaces.

## Evidence
- Docs: [Remote control](https://code.claude.com/docs/en/remote-control)
- Docs: [Claude Code overview](https://code.claude.com/docs/en/overview)
- The current docs position remote control around:
  - checking on active local runs from elsewhere
  - sending follow-up instructions into those runs
  - continuing the same coding thread across devices instead of starting over

## Product signal
Claude Code is moving toward ambient continuity, where the active coding run matters more than the specific surface where it started.
