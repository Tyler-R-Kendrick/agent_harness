# Web And IDE Continuity

- Harness: Claude Code
- Sourced: 2026-05-22

## What it is
Claude Code spans terminal, browser, and IDE surfaces so the same harness can move between local editing, web supervision, and editor-centric review.

## Evidence
- Docs: [Claude Code overview](https://code.claude.com/docs/en/overview)
- The current overview and integration docs call out:
  - browser-based Claude Code with no local setup path
  - IDE integrations for editors such as VS Code and JetBrains
  - long-running tasks and parallel work that can be supervised outside the terminal
  - continuity across local and remote surfaces

## Product signal
Multi-surface continuity is part of the harness strategy, not a bolt-on convenience.
