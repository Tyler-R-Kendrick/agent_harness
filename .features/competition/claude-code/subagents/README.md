# Subagents

- Harness: Claude Code
- Sourced: 2026-05-22

## What it is
Claude Code supports specialized subagents that each get their own context, prompts, and permission profile so users can decompose work by role.

## Evidence
- Docs: [Subagents](https://code.claude.com/docs/en/sub-agents)
- Key properties:
  - separate context window
  - specialized expertise
  - reusable/shareable configs
  - flexible per-agent tool permissions
  - user-defined system prompts per subagent

## Product signal
Anthropic is explicitly decomposing the harness into a small agent system rather than one monolithic assistant.
