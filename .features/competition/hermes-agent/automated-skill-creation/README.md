# Automated Skill Creation

- Harness: Hermes Agent
- Sourced: 2026-04-24

## What it is
When Hermes solves a hard problem, it writes a reusable skill document for later reuse.

## Evidence
- Official site: [Hermes Agent](https://hermes-agent.org/)
- The site states that created skills are searchable, shareable, and compatible with the `agentskills.io` standard.

## Product signal
Hermes treats self-improving workflow capture as a first-class mechanism.
