# Multi-Channel Gateway

- Harness: Hermes Agent
- Research date: 2026-04-23
- What it does: Hermes runs through a single gateway process that serves CLI plus Telegram, Discord, Slack, WhatsApp, Signal, and other messaging surfaces.
- Why it matters: Messaging-native control turns the agent into an always-available background worker instead of an IDE-only tool.
