# Multi-Model Routing

- Harness: Hermes Agent
- Research date: 2026-04-23
- What it does: Hermes supports many providers and lets users switch models with `hermes model` without changing code. Recent releases also added fast mode and provider-specific routing improvements.
- Why it matters: Flexible model routing lowers lock-in and makes a harness more resilient to pricing, latency, and capability shifts.
