# Scheduled Automations

- Harness: Hermes Agent
- Research date: 2026-04-23
- What it does: Hermes includes a built-in cron scheduler so the agent can run recurring tasks like reports, audits, and backups without a live conversation.
- Why it matters: Native scheduling is a strong differentiator versus harnesses that only operate in foreground sessions.
