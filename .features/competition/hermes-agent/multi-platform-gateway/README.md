# Multi-Platform Gateway

- Harness: Hermes Agent
- Sourced: 2026-04-24

## What it is
Hermes can operate through Telegram, Discord, Slack, WhatsApp, Signal, and CLI from one gateway.

## Evidence
- Official site: [Hermes Agent](https://hermes-agent.org/)
- The site highlights cross-platform continuation and voice memo transcription.

## Product signal
Hermes is optimized for ambient access rather than a single local interface.
