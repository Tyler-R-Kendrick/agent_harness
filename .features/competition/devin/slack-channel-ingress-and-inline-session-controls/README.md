# Slack Channel Ingress And Inline Session Controls

- Harness: Devin
- Sourced: 2026-05-02

## What it is
Devin's Slack integration lets teams launch and follow sessions from chat, which extends the harness into the ambient channel where work is often requested and reviewed.

  - the January 14, 2026 release notes mention Slack session permalinks, which signals ongoing investment in durable chat-side handoff and navigation
