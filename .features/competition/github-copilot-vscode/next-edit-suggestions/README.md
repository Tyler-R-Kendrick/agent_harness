# Next Edit Suggestions

- Harness: GitHub Copilot in VS Code
- Research date: 2026-04-23
- What it does: Next edit suggestions predict where the next code change should happen and offer the completion, with gutter UI and keyboard navigation.
- Why it matters: Predictive multi-location editing is one of the most concrete near-term UX upgrades beyond autocomplete.
