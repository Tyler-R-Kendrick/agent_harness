# Enterprise Global Skills And Remote Config

- Harness: Cline
- Sourced: 2026-04-30

## What it is
Recent Cline releases add enterprise-managed skills and remote configuration, moving capability packaging and onboarding policy into centrally controlled product surfaces.

## Evidence
- Official release page: [cline/cline releases](https://github.com/cline/cline/releases)
- Latest development checkpoints:
  - `v3.81.0`, dated April 24, 2026, added GPT-5.5 model support for OpenAI Codex subscription users and improved runtime memory diagnostics
  - `v3.80.0`, dated April 22, 2026, wired remote `globalSkills` into enterprise remote config with UI toggles, an `Enterprise Skills` section, and `alwaysEnabled` enforcement
  - `v3.79.0`, dated April 16, 2026, added `globalSkills` to remote config and Azure Blob Storage as a storage provider

## Product signal
This is a clear move from individual-user customization toward centrally managed workflow distribution, policy, and fleet-level runtime tuning.
