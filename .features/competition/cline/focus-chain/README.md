# Focus Chain

- Harness: Cline
- Sourced: 2026-04-30

## What it is
Focus Chain is Cline's built-in long-task tracker: it creates and updates a visible checklist that persists across context compaction.

## Evidence
- Official docs: [Focus Chain](https://docs.cline.bot/features/focus-chain)
- First-party details:
  - the docs describe it as automatic todo list management with real-time progress tracking
  - the checklist persists across context window resets
  - Cline stores the todo list as an editable markdown file
  - the task header shows progress counts such as `3/8`
  - the page includes a product screenshot for the task tracker UI

## Product signal
This is a concrete response to long-running task drift: visible progress state becomes part of the harness UX instead of staying inside the model's private scratchpad.
