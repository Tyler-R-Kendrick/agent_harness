# MCP Marketplace And Server Builder

- Harness: Cline
- Sourced: 2026-04-30

## What it is
Cline treats MCP as a first-class extension layer with marketplace distribution, CLI support, and agent-assisted server creation.

## Evidence
- Official docs: [MCP Overview](https://docs.cline.bot/mcp/mcp-overview)
- First-party details:
  - the docs position MCP as the standard way to connect models to tools and data sources
  - users can install servers from the Cline MCP Marketplace
  - MCP works in both the VS Code extension and the CLI
  - the docs say Cline can help build MCP servers by cloning repos, handling config, and troubleshooting
  - the integration model includes dynamic tool suggestions and combining multiple MCP servers in one task

## Product signal
Cline is reinforcing the trend that serious harnesses need a secure extension bus, not just a closed set of built-in tools.
