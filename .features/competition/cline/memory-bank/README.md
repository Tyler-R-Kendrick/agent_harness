# Memory Bank

- Harness: Cline
- Sourced: 2026-04-30

## What it is
Cline promotes a structured markdown-based Memory Bank so project context survives across sessions and context-window resets.

## Evidence
- Official docs: [Memory Bank](https://docs.cline.bot/features/memory-bank)
- First-party details:
  - the docs call it a structured documentation system that helps Cline maintain context across sessions
  - it is presented as a way to turn Cline from a stateless assistant into a persistent development partner
  - the default structure includes core project, product, architecture, tech, and active-context files
  - the workflow explicitly pairs Memory Bank with Plan mode when resuming work

## Product signal
Cline treats durable project memory as a user-visible operating model, not just a hidden retrieval layer, which reinforces the market shift toward inspectable layered memory.
