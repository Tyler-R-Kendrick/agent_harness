# Research Workspace

This directory stores paper-by-paper research packets.

## Structure

- Create one subfolder per paper/topic using: `<slug>-<id>` (example: `continual-harness-2605.09998`).
- Each paper folder should include:
  - `README.md` — summary, key ideas, and implementation notes.
  - `reference-architecture.md` — system design and integration points.
  - `experiments/` — reproducible implementation attempts and logs.
